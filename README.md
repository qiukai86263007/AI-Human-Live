# AI-Human-Live


## 软件介绍

`AI-Human-Live` 是一个AI数字人直播系统，小白也可使用。 支持视频合成、声音合成、声音克隆，简化本地模型管理、一键导入和使用AI模型。

## 技术栈

- `electron`
- `vue3`

## 本地运行开发

> 仅在 node 20 测试过

```shell
# 安装依赖
npm install
# 调试运行
npm run dev
# 打包
npm run build
```
